# Customer data safety guide

- Use a supported browser on an encrypted, screen-locked device.
- Do not use private/incognito mode for care records.
- Confirm cloud sync after important edits.
- Download a password-encrypted complete backup regularly.
- Keep the backup password separate from the backup file.
- Test restoring a backup before an emergency.
- Revoke a caregiver’s account immediately when access is no longer needed.
- Do not share workspace URLs or calendar subscription tokens publicly.
- Printed handovers and exported calendars may contain sensitive information; store and dispose of them securely.
- CarePlan assists organisation and communication. Always follow the current clinician-approved care and medication plans.
